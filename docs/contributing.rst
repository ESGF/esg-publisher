Contributing
============

 * Please document your pull requests so we can understand how to test your changes.  We don't want changes to affect publishing of ongoing projects.
 * Please **DO NOT** make pull requests to the ``main`` branch.  Please make pull requests to the ``integration`` branch so we can test your changes before merging to ``main``.  We may choose to close your PR to ``main`` and you will need to re-open it to ``integration``.

Updates to this document
------------------------

Please install the Sphinx package.  Also you will need to `pip install sphinx-glpi-theme` in your environment.
